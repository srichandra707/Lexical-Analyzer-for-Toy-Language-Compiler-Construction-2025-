To install and use flex, You need to install MinGW C compiler, Use this video to install it:

Download MinGW: https://www.mingw-w64.org/downloads/

1. Install MinGW to 'C:\MinGW'
2. Add 'C:\MinGW\bin' to the system's PATH environment variable.
   - type 'env' in search and then you'll get to the tab of env variables
   - Navigate to the 'Advanced tab', then click 'Environment Variables'
   - Find the 'Path' variable under 'System Variables', click 'Edit', and add `C:\MinGW\bin`

https://www.youtube.com/watch?si=bZ70fNvSTUkP1wKe&v=JsO58opI3SQ&feature=youtu.be

Note: On the final step where this guy opens the MinGW installation manager, make sure 'msys-flex bin' type is selected.
